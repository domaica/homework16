Poverty vs Healthcare. Correlation = 0.5245

Positive or linear correlation. As poverty increases, there is a bigger % of population with lack of healthcare. Correlation is slightly above 0.5, therefore we can conclude that it implies a moderate correlation between poverty and lack of healthcare in different US states.

Poverty vs Obesity. Correlation = 0.4623

There is a linear correlation positive. It means that when poverty increases, so does the obesity. Anyway, it is a weak correlation as value is smaller than 0.5.

Poverty vs Smoking. Correlation = 0.5208

As poverty increases, there is a bigger amount of smokers. Correlation is slightly above 0.5 so the interpretation for this implies a moderate correlation between poverty and smoking.

Age vs Obesity. Correlation = -0.0077

The correlation value is so close to zero that it can be considered that there is no correlation between these 2 factors.

Age vs Smoking. Correlation = 0.2481

As age increases there is a trend towards increasing smoking, but correlation value is aproximately 0.25 so the correlation is weak.

Age vs Healthcare. Correlation = -0.3852

This correlation value is negative or non linear what implies that as one factor (age) increases, the other (Healthcare) decreases, but it is a weak trend again.

Household Income vs Obesity. Correlation = -0.6511

This correlation value is negative or non linear what implies that as one factor (household income) increases, the other (obesity) decreases. In addition to that, correlation is a value close to -0.7 and bigger than -0.5, so we can conclude that there is a moderate, almost strong negative correlation between income and obesity. The bigger the income, obesity decreases.

Household Income vs Smoking. Correlation = -0.6601

Same as above. Negative correlation between income and smoking. The bigger the household income in a given US State, residents there tend to smoke less.

Household Income vs Healthcare. Correlation = -0.4340
